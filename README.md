# The Sovereign Governance - Makronesia State to Act-Ark
## Project ID: -STG-INTEGRATION

Monorepo ini menyatukan sistem pengambilan keputusan kolektif ($QSTATE) 
dengan otoritas Veto Sovereign Government (STG).

### Struktur Integrasi:
- **apps/qstate-engine**: Algoritma penghitung Quorum.
- **apps/stg-veto-portal**: Modul eksekusi Veto Akhir.
- **packages/shared-assets**: Protokol komunikasi antar-modul.
